# JavaScript Fundamentals + Live Digital Clock

Core JavaScript concepts including loops, arrays, objects, and error handling — plus a live digital clock built with DOM manipulation and `setInterval`.

## Highlights

### Live Digital Clock
`index.html` + `data.js` — a real-time clock that updates every second using `Date.toLocaleTimeString()` and `setInterval()`. A clean example of JavaScript timers and DOM updates working together.

### Core Concepts
| File | Topic |
|---|---|
| `forloop1.js` | For loops and iteration |
| `dowhile1.js` | Do-while loop patterns |
| `array3.js` / `forarray1.js` | Array traversal with loops |
| `object1.js` | Object creation and access |
| `error2.js` / `arrayerror1.js` | Error handling patterns |
| `jsfub.js` | Functions and scope |
